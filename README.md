  # Game Chat Server                                          
                                                              
  Real-time multiplayer chat server built with Node.js and WebSocket.                
                  
  ## Tech Stack

  - Node.js                                                   
  - WebSocket (ws)
  - Express                                                   
                  
  ## Features

  - Real-time messaging
  - Multiple chat rooms
  - Multi-user support
                                                              
  ## Getting Started
                                                              
  ### Install dependencies
  \```bash
  npm install
  \```

  ### Run server
  \```bash
  node server.js
  \```
                                                              
  ### Open client
  Go to `http://localhost:3000` in your browser.              
  Open multiple tabs to test multi-user chat.
                                                              
  ## Project Structure
                                                              
  \```            
  game-chat-server/
  ├── client/
  │   └── index.html   # Test client
  ├── server.js        # WebSocket server                     
  └── package.json
  \```                                                        
                  
  ## Why This Project                                         
   
  Built to learn real-time server architecture using WebSocket
   —              
  a core technology used in multiplayer game servers.
